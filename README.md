KernelForge

Android kernel tuning and system monitoring tool for rooted devices.

Overview

KernelForge provides a set of tools to adjust kernel parameters and monitor system status in real time.
It is intended for advanced users who want more control over device performance and behavior.

Features

Kernel Tweaks

- CPU governor configuration
- I/O scheduler selection
- VM parameter tuning
- TCP congestion control
- ZRAM toggle

System Monitoring

- CPU usage
- RAM usage
- Battery status
- Thermal information

System Tools

- Boot apply settings
- Root-based system controls

Requirements

- Android 8.0+
- Root access (Magisk / KernelSU recommended)

Installation

Clone the repository:

git clone https://github.com/lenirraxzone/KernelForge.git

Build using Android Studio or Gradle.

Notes

- Not all features are supported on every device
- Incorrect configuration may cause instability
